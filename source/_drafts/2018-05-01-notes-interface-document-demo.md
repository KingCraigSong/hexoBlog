## 接口信息

### 用户设置

#### 新增

1. 请求地址：http://www.baidu.com
1. 请求方法：POST
1. 请求参数示例：

```http
id=6&nameLike=张三
```

1. 请求头：

```http
Content-Type: application/json;charset=UTF-8
```

1. 请求体示例：

```json
  {
    "rearNo": "",
    "feeName": "后勤报销测试",
    "feeCode": "2",
    "unit": "2",
    "amount": "6",
    "price": "10",
    "totalPrice": "60",
    "rearBy": "莫",
    "orgId": "depart2",
    "convoyId": "CD-SCCD",
    "rearDate": "2018-05-04 00:00:00",
    "remark": ""
  }
```

1. 请求参数说明：

| 参数名 | 类型    | 必填 | 默认值 | 说明 |
| ------ | ------- | ---- | ------ | ---- |
| rearNo | Integer | Y    |        | 编号 |

1. 返回结果示例：

```json
  {
    "stateCode": 200,
    "state": "请求成功"
  }
```

1. 返回结果参数说明：

| 参数名 | 类型    | 必填 | 默认值 | 说明 |
| ------ | ------- | ---- | ------ | ---- |
| rearNo | Integer | Y    |        | 编号 |
