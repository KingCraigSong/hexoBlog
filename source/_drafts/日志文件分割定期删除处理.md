# tomcat 日志处理

## 分割、删除 N 天前日志

```bash
#!/bin/bash

DIR=日志地址
d=`date +%Y%m%d`
cd $DIR
cp catalina.out catalina.out.bak${d}
echo "" > catalina.out

find ./ -mtime +7 -name "catalina.out.bak*" -exec rm -rf {} \;
```

## 添加定时任务

将auto_clean.sh执行脚本加入到系统计划任务，到点自动执行

```bash
crontab -e
```

输入：

```ini
59 23 * * * /opt/soft/log/auto_clean.sh >/dev/null 2>&1
```

这里的设置是每天23点59分执行auto_clean.sh文件进行数据清理任务了。
